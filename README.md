# [pentestX](https://github.com/furious-05/pentestX)

**pentestX** is a lightweight web tool for penetration testers that speeds up command construction and keeps all frequently-used commands accessible in one place.

A simple description:

* Fast, interactive command builder — fill inputs and get ready-to-run commands.
* Left sidebar: your category-specific options and commands (quick select).
* Right sidebar: global categories for quick navigation.
* Dark / Light theme, responsive layout, and copy-to-clipboard for commands.


## Features (short)

* Template-based command generation with input fields
* Instant copy-to-clipboard for commands
* Global search with suggestions (navigate directly to category / option)
* Persistent, resizable left/right sidebars for commands and categories
* Dark / light mode and mobile-friendly UI
